Privacy Policy — Cosmic Match

Last updated: April 2026

Cosmic Match is a puzzle game. This policy explains what data is collected and how it is used.

Data stored on your device

Your game scores and progress are stored locally on your device using an encrypted database. This data is never transmitted to any external server.

No personal data collected

Cosmic Match does not collect, store, or transmit any personally identifiable information. No account is required to play.

Crash reporting

If a Sentry DSN is configured at build time, anonymous crash reports (stack traces, device model, OS version) may be sent to Sentry.io. No game data or personally identifiable information is included in crash reports.

Feedback (optional)

If you use the in-app feedback feature, a description and optional screenshot are submitted to the developer. This is always user-initiated.

Internet permission

The app requests internet access solely for optional crash reporting and feedback submission. The game itself works fully offline.

Third-party services

  • Sentry.io — optional crash reporting. Subject to Sentry's privacy policy.

Data deletion

Uninstalling the app removes all locally stored game data.

Contact

Questions about this policy can be directed to the developer via GitHub Issues on the project repository.
