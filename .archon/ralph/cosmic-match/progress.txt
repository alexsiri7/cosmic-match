## Codebase Patterns

### Flutter project structure
- **Where**: project root
- **Pattern**: Flutter project at repo root (not nested), standard `flutter create` layout
- **Example**: `lib/main.dart` → `CosmicMatchApp`, `lib/game/cosmic_match_game.dart` → `CosmicMatchGame extends FlameGame`

### Validation commands
- **Where**: CLI
- **Pattern**: Flutter SDK at `/tmp/flutter-sdk/bin`, use `export PATH="/tmp/flutter-sdk/bin:$PATH"` before commands
- **Example**: `flutter analyze`, `flutter test`
